• Include at least one semantic long tail signal

GOOD

BAD

SEMANTIC ANCHOR REQUIREMENT

Each category paragraph must include at least ONE strong gameplay anchor.

Examples  
Platform → jumping, obstacles, levels  
FPS → shooting, aiming, combat  
Puzzle → solving, matching, logic  
Animal → caring, exploring, guiding  
Vehicle → driving, racing, control  
Simulation → managing, building, operating  

KEY FEATURES SECTION RULES

Title must be exactly:

• Include 4 to 6 bullets  
• Ideal bullet length: 4 to 7 words  
• Bullets must be gameplay entity phrases  
• No marketing language  
• No filler adjectives  
• Avoid repeating category name  
• Avoid full sentences  

BULLET LINGUISTIC STRUCTURE

GOOD

BAD

BULLET SEMANTIC DISTRIBUTION MODEL

Each category bullet set should cover:
