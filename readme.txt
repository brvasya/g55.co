MASTER CATEGORY DESCRIPTION GENERATOR RULE V2 (EDITORIAL SEO OPTIMIZED)

Goal
Generate handcrafted editorial quality category descriptions for G55.CO that maximize SEO authority, semantic clarity, and readability.
Descriptions must resemble high quality gaming portal editorial content.

INPUT VARIABLES
- Category name: {CATEGORY_NAME}
- Category type: {GENRE | MECHANIC | THEME | VEHICLE | CLASSIC | CASUAL | TREND}
- Audience: casual gamers, teens, kids, desktop users
- Tone: friendly, expert, concise, editorial

OUTPUT STRUCTURE (STRICT)

1 Intro paragraph  
2 "Key Features" section  
3 Bullet list

INTRO PARAGRAPH RULES

• Write exactly ONE paragraph  
• Length: 60 to 90 words  
• First sentence MUST start with gameplay action  
• Clearly describe what players do in this category  
• Mention core mechanics naturally  
• Include at least one semantic long tail signal  
• Avoid excessive soft adjectives  
• Avoid marketing tone  
• Avoid filler phrases  
• Avoid formal or academic style  
• Do not use bold text  
• Do not use dashes in prose  
• Must sound like human gaming portal editor  

FIRST SENTENCE GAMEPLAY ACTION RULE

Start with verbs or gameplay focus.

GOOD  
Animal Games focus on caring for pets and guiding wildlife  

BAD  
Animal Games place players in playful worlds  

SEMANTIC ANCHOR REQUIREMENT

Each category paragraph must include at least ONE strong gameplay anchor.

Examples  
Platform → jumping, obstacles, levels  
FPS → shooting, aiming, combat  
Puzzle → solving, matching, logic  
Animal → caring, exploring, guiding  
Vehicle → driving, racing, control  
Simulation → managing, building, operating  

KEY FEATURES SECTION RULES

Title must be exactly:

Key Features

• Include exactly 5 bullets  
• Ideal bullet length 3 to 6 words  
• Bullets must be gameplay entity phrases  
• No marketing language  
• No filler adjectives  
• Avoid repeating category name  
• Avoid full sentences  

BULLET LINGUISTIC STRUCTURE

Prefer noun based mechanic phrases.

GOOD  
Pet care interaction mechanics  
Exploration driven character gameplay  

BAD  
You can take care of pets  
Players will explore many worlds  

BULLET SEMANTIC DISTRIBUTION MODEL

Each category bullet set should cover:

1 Core mechanic  
2 Player skill or action  
3 Mode or system  
4 Theme or scenario  
5 Optional technical context  

CATEGORY TYPE SEMANTIC GUIDANCE

GENRE  
Focus on gameplay loop and player goals  

MECHANIC  
Focus on systems and actions  

THEME  
Focus on characters, world, scenario  

VEHICLE  
Focus on control, physics, speed  

CLASSIC  
Focus on rules, logic, timeless play  

CASUAL  
Focus on accessibility and simplicity  

TREND  
Focus on scenario and cultural context  

SOFT ADJECTIVE CONTROL RULE

Limit use of words such as:

fun  
playful  
exciting  
friendly  
lighthearted  

Use maximum 1 or 2 per paragraph.

SEMANTIC ENRICHMENT RULE

Include at least one long tail gameplay entity when natural.

Examples  
Sandbox building mechanics  
Real time multiplayer arenas  
Turn based tactical systems  

STYLE MODEL

Write like concise professional portal editor.  
Not blog.  
Not ad copy.  
Not generic AI template.

• feature

Before output ensure:
